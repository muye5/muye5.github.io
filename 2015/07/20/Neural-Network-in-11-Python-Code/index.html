Neural Network in 11 Python Code

原文章
看到这个11行搞定Neural Netwok的Python代码，禁不住转了过来，就喜欢这种短小精悍却又思路清晰的Code，直接上代码：

  • 这个NN的功能是实现异或操作
  • 4个训练样本，3维，最后一维是 bias，输入始终为1；
  • 网络结构是 (3-4-1)，激活函数是 sigmoid 函数，最后的输出节点也使用了激活函数，有时候最后的输出不再进行变换，error的计算会有些不同；
  • syn0 是输入层到隐层节点的权重矩阵，syn1 是隐层到输出层的权重矩阵，l1是隐层节点的 sigmoid 输出，l2 就是预测结果；
  • l2_delta 是在对 np.dot(l1, syn1) 求导，其中(l2 * (1 - l2)) 来自 sigmoid，l1_delta 是在对 np.dot(X, syn0) 求导；
  • l1.T.dot(l2_delta) 是对 syn1 求导，X.T.dot(l1_delta) 是对 syn0 求导，因为 syn0 和 X 是一个dot操作，所以求导的时候就直接剩下 X 这个系数了。

代码很简练，可能不直观的地方在于那几次转置，画画图就好了，简记。
